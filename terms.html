Användarvillkor

Dessa användarvillkor (”Villkoren”) reglerar användningen av BotJahl – en AI-chattbotlösning som tillhandahålls av Westjahl Creative AB (”Leverantören”, ”vi”, ”oss”). Genom att registrera ett konto, starta provperiod, betala för abonnemang eller på annat sätt använda tjänsten godkänner du dessa Villkor. Dessa villkor inkluderar personuppgiftsbiträdesavtalet (DPA) som finns på denna sida.

Senast uppdaterad: 2025-11-20

1. Om tjänsten

BotJahl är en molnbaserad SaaS-tjänst där företagskunder (”Kunden”) kan skapa en AI-chattbot som bäddas in på Kundens egen webbplats. Tjänsten gör det möjligt att:

  • registrera ett företagskonto med kund-ID,
  • ladda upp material (t.ex. PDF, DOCX, TXT, MD) eller indexera befintliga webbsidor,
  • låta en AI-chattbot besvara frågor baserat på Kundens material samt allmän/domänspecifik kunskap.

Tjänsten drivs tekniskt med hjälp av OpenAI:s API (AI-modeller), Microsoft Azure (molninfrastruktur) och Stripe (betalningar), samt en extern e-posttjänst för kommunikation.

2. Konto, registrering och åtkomst

För att använda tjänsten måste Kunden registrera ett konto. Vid registrering behöver följande uppgifter lämnas:

  • företagsnamn,
  • e-postadress,
  • webbplats-URL,

Kunden ansvarar för att de uppgifter som lämnas är korrekta och uppdaterade. Inloggning till adminportalen sker med kund-ID och den e-postadress som angavs vid registrering. Kunden ansvarar för att hålla inloggningsuppgifter hemliga och för alla aktiviteter som sker via Kontot.

3. Licens till kundinnehåll

Kunden behåller äganderätten till allt innehåll som laddas upp eller indexeras via tjänsten (”Kundinnehåll”), t.ex. dokument, texter och webbsidor.

Kunden ger Leverantören en begränsad, icke-exklusiv, icke överlåtbar licens att lagra, bearbeta och använda kundinnehåll endast i följande syften:

  • att extrahera text ur filer och webbsidor,
  • att skapa vektorrepresentationer (”embeddings”) av innehållet,
  • att använda dessa embeddings för att generera svar i kundens AI-chattbot,
  • att tillhandahålla, drifta och förbättra funktionen inom Kundens eget konto.

Kundinnehåll används endast för kundens egen bot och delas inte med andra kunder. Leverantören har inte rätt att sälja, publicera eller använda kundinnehåll för andra ändamål än att leverera tjänsten.

Kunden ansvarar för att denne har nödvändiga rättigheter (t.ex. upphovsrätt, licenser och tillstånd) till allt kundinnehåll som laddas upp eller indexeras via tjänsten.

4. Acceptabel användning och kundens ansvar

Kunden ansvarar för all användning av tjänsten som sker genom Kundens konto samt för allt kundinnehåll. Kunden förbinder sig att inte använda tjänsten för att:

  • bryta mot gällande lagstiftning, inklusive men inte begränsat till upphovsrätt, marknadsförings- och dataskyddslagstiftning,
  • sprida olagligt, kränkande, diskriminerande, rasistiskt, pornografiskt eller på annat sätt olämpligt material,
  • uppmana till eller främja våldsbrott, bedrägeri, trakasserier eller annan brottslighet,
  • innehålla skadlig kod, spam eller annan skadlig aktivitet.

Kundens besökare/slutanvändare interagerar med AI-chattboten via Kundens webbplats. Kunden ansvarar för:

  • hur chattbotens svar presenteras och används,
  • att informera sina besökare om eventuella begränsningar i svarens riktighet,
  • att tillhandahålla egna villkor och integritetspolicy för sin webbplats.

Leverantören förbehåller sig rätten att tillfälligt stänga av eller avsluta konton som misstänks bryta mot dessa Villkor eller gällande lag.

5. Betalning, provperiod och abonnemang

Tjänsten kan erbjudas med en kostnadsfri provperiod under begränsad tid, enligt vad som framgår vid registreringen. Efter provperiodens slut övergår tjänsten till ett betalt abonnemang om kunden lägger in betalmetod och slutför betalning via Stripe.

Betalningar hanteras av Stripe, och genom att ange betalningsuppgifter godkänner Kunden Stripes egna villkor och integritetspolicy. Leverantören lagrar inte kundens fullständiga kortuppgifter.

Abonnemanget löper normalt månadsvis (eller enligt annan angiven period) och förnyas automatiskt fram till dess att det sägs upp av Kunden. Kunden kan säga upp abonnemanget när som helst via kundportalen. Uppsägning gäller från nästa faktureringsperiod och ingen återbetalning ges för redan betald period om inget annat uttryckligen avtalats.

Vid utebliven betalning kan Leverantören begränsa tillgången till tjänsten eller stänga av kontot.

6. AI-svar, upphovsrätt och ansvarsbegränsning

AI-chattbotens svar genereras automatiskt med hjälp av OpenAI:s modeller och kundens eget material. Svaren kan innehålla felaktigheter, föråldrad information eller tolkningar. Leverantören lämnar inga garantier för riktighet, fullständighet eller lämplighet för ett visst ändamål.

Kunden ansvarar för att granska och, vid behov, korrigera innehåll som presenteras för slutanvändare. Kunden bör inte använda chattboten som enda källa för juridiska, medicinska, finansiella eller andra kritiska beslut.

I den mån det är tillåtet enligt tvingande lag ansvarar leverantören inte för:

  • indirekta skador, följdskador eller utebliven vinst,
  • förlust av data eller affärsmöjligheter,
  • skador orsakade av fel eller driftstörningar i tredjeparts-tjänster (t.ex. OpenAI, Azure, Stripe, e-posttjänster).

Leverantörens sammanlagda ansvar under en rullande tremånadersperiod är begränsat till det totala belopp som kunden betalat till leverantören för tjänsten under samma period, om inte annat följer av tvingande lag.

7. Uppsägning och radering av data

Kunden kan när som helst avsluta sitt konto genom att säga upp abonnemanget i kundportalen eller kontakta Leverantören. Leverantören kan säga upp avtalet med skälig uppsägningstid, eller om Kunden väsentligen bryter mot dessa Villkor.

Vid uppsägning:

  • stängs åtkomst till adminportalen när abonnemangsperioden löpt ut,
  • tas chattboten ur funktion på Kundens webbplats (API-anrop nekas),
  • raderas uppladdade dokument och embeddings normalt inom 30 dagar,
  • kan viss information (t.ex. fakturaunderlag och loggar) sparas längre om det krävs enligt lag eller berättigat intresse (t.ex. för bokföring och säkerhet).

8. Ändringar av tjänsten och villkoren

Leverantören kan när som helst uppdatera tjänsten, t.ex. genom att lägga till eller ta bort funktioner, byta teknisk infrastruktur eller justera prissättning. Väsentliga ändringar med negativ påverkan för kunden kommuniceras i förväg, om möjligt.

Leverantören kan även uppdatera dessa Villkor. Den senaste versionen hålls tillgänglig på webbplatsen. Vid väsentliga ändringar informeras Kunden via e-post eller i samband med inloggning. Fortsatt användning av tjänsten efter att ändrade villkor trätt i kraft innebär att Kunden accepterar dessa.

9. Tillämplig lag och tvistlösning

Detta avtal regleras av svensk lag. Tvister som uppstår i anledning av dessa Villkor eller användningen av tjänsten ska i första hand lösas genom dialog mellan parterna.

Om tvisten inte kan lösas genom förhandling ska den avgöras av svensk allmän domstol med Stockholm tingsrätt som första instans, om inte annat följer av tvingande lag.
